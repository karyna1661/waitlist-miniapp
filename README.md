# Let's Connect Waitlist Mini-App

This is a standalone Farcaster mini-app for the Let's Connect waitlist.

## Deployment

1. Copy this folder to a new location
2. Run `npm install` 
3. Set environment variables in `.env.local`:
   ```
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
   NEYNAR_API_KEY=your_neynar_api_key
   ```
4. Deploy to Vercel

## Farcaster Registration

Use domain: `letsconnect-waitlist.vercel.app` (no path needed)
